**Homepage Kit**  
  
このリポジトリは、ホームページ制作時に繰り返し利用する機能をまとめた個人用の再利用キットです。学習成果の整理と、同じ機能を再度作成する際の作業短縮を目的としています。  
***
**目的**  
  
・学習成果・実装経験をまとめて管理する  
・ハンバーガーメニューやフィルターなど、よく使うパーツをコピペで即利用できるようにする  
・機能を追加しながら自分用の "部品倉庫" を育てる  
***
**構成方針**  
  
本リポジトリは 1つのリポジトリに機能をフォルダ分割してまとめています。これにより、管理が簡単になり、必要な機能だけをフォルダごとコピーして使うことができます。
